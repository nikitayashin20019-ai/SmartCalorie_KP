Описание
Десктопное приложение для подсчёта калорий. Позволяет вручную добавлять еду из справочника, добавлять целые рецепты одним кликом и распознавать еду по фотографии через ИИ.

Стек
Python 3.10+
GUI: tkinter (встроенный)
Сеть: requests (для OpenRouter API)
Данные: JSON файлы

Архитектура
gui.py — отвечает только за интерфейс (вкладки ttk.Notebook). Не содержит бизнес-логики.
database.py — абстракция над JSON файлами. Читает справочники, пишет в user_data.json.
ai_scanner.py — изолированный модуль для работы с API OpenRouter. Кодирует картинку в Base64, парсит JSON ответ ИИ.
data/ — статические файлы (справочники), не изменяются пользователем.
Особенности парсинга ИИ
ИИ часто возвращает JSON обёрнутый в markdown блоки (json ... ). В ai_scanner.py обязательно нужна очистка строки от этих тегов перед json.loads().
